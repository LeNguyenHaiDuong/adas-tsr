# Glossary — TSR Documentation

| Thuật ngữ | Nghĩa chuẩn trong repo |
|---|---|
| `Prototype` | Hệ thống giữa kỳ hoặc baseline repo đang chạy thật |
| `Production` | Feature TSR cấp xe, không đồng nghĩa với detector |
| `Detector output` | Bbox + class/confidence theo frame |
| `Feature output` | Sign/state đã qua temporal/context/state logic |
| `HMI output` | Thông tin cuối cùng được phép hiển thị/cảnh báo cho người lái |
| `Overlay` | Video annotation phục vụ demo, không phải HMI thật |
| `Temporal filtering lite` | Logic đơn giản hiện dùng trong prototype trước khi có tracking/state manager đầy đủ |
| `State Manager` | Khối quản lý lifecycle sign rõ ràng qua nhiều frame |
| `ODD` | Operational Design Domain |
| `SOTIF` | Safety of the Intended Functionality |
| `JSONL` | Machine-readable log, thường dùng cho replay/RCA/regression |
